# Intelligence Prompts Directory

A community-driven, open-source directory for high-performance LLM system prompts. Built with **11ty** and **Tailwind CSS**.

![Prompts Directory Screenshot](https://via.placeholder.com/800x400) <!-- You can update this with a real screenshot later -->

## Features

- **"Prompts as Code"**: All prompts are stored as Markdown files in `src/prompts/`.
- **Search & Filter**: filtering to find the right prompt for your model.
- **Explore**: Browse by [Authors](/authors/), [Categories](/categories/), and [Tags](/tags/).
- **Modern UI**: Sleek, dark-mode design focused on readability.
- **Copy-Paste Ready**: One-click copy for all system prompts.

## Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/iqprompts/iqprompts.git
    cd iqprompts
    ```

2.  **Install dependencies**:
    ```bash
    pnpm install
    ```

3.  **Start the development server:**
    ```bash
    pnpm run dev
    ```
    Visit `http://localhost:8080` to see the site.
